    # Modal 对话框

    # API

    点击模态框的蒙层和右上角关闭图标时会关闭对话框，即会更改传入的visible。所以这里的visible要用.sync修饰符来绑定

    参数 说明 类型(参数) 默认值
    visible 对话框是否可见，需要使用.sync修饰符 boolean false
    title 对话框标题 string _
    width 对话框宽度 string 50%
    cancelText 取消按钮文字 string 取消
    okText 确认按钮文字 string 确认
    customFooter 是否自定义页脚 boolean false
    on-cancel 点击取消按钮时触发 _ _
    on-ok 点击确认按钮时触发 _ _
